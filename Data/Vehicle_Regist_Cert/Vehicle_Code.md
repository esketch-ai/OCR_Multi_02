# **차대번호(VIN) 17자리 각각에 담긴 의미**

차대번호(VIN)는 아래와 같이 3개의 구역으로 나뉘며 각 자리에 고유한 정보가 담겨 있습니다.

  

|   |   |   |
|---|---|---|
|**구분**|**자리**|**의미**|
|WMI|1~3자리|차량의 제조국, 제조사, 차종|
|VDS|4~9자리|차량 종류, 차체 형식, 엔진 등 기술 정보|
|VIS|10~17자리|모델 연식, 생산 공장, 일련번호|

  

그럼 3개의 구역에 어떤 정보가 담겨 있는지 더 자세히 살펴볼까요?

  

### **✔️ WMI (World Manufacturer Identifier)**

-    1번째 자리: 제조 국가

-    2번째 자리: 제조사

-    3번째 자리: 차량 구분

  

### **✔️ VDS (Vehicle Descriptor Section)**

4~8번째 자리는 차종, 엔진, 트랜스미션 형식 등 차량의 사양을 나타내며, 9번째 자리는 차대번호(VIN)의 유효성을 확인하는 검증 코드입니다.

  

### **✔️ VIS (Vehicle Identifier Section)**

-    10번째 자리: 제조년도

알파벳과 숫자를 섞어 30년 주기로 돌아가는데, 1980년을 기준으로 1년마다 알파벳 순서대로 A~Y 코드로 나열됩니다. 알파벳이 끝나면 1~9까지의 숫자 코드가 부여됩니다. 다만, 다른 숫자와 알파벳으로 혼동하거나 조작 가능성이 있는 I, O, Q, U, Z는 사용하지 않습니다.

  

|   |   |
|---|---|
|**연식**|**코드**|
|2021|M|
|2022|N|
|2023|P|
|2024|R|
|2025|S|
|2026|T|

  

-    11번째 자리: 생산 공장 코드

-    12~17자리: 차량 일련번호
